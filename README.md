# 网易云音乐客户端

## 项目技术栈：

- electron V19.0.4
- Vue V3.0
- Vite V2.9.9
- Vuex
- lodash
- element-Plus
- axios
- dayjs

---
## 运行
依赖安装

```shell
yarn add global concurrently
```

运行

```shell
yarn 
yarn electron:serve
```

打包

```shell
yarn electron:build
```
